Scrivi un programma in C++ che rispetti le seguenti specifiche.

1. Crea una struct chiamata Sensore che contenga tre variabili:
  • double temperatura
  • int segnalazioni
  • bool guasto
2. Definisci due variabili globali:
  • int totaleSegnalazioni = 0;
  • const double LIMITE = 75.0;
3. Crea le seguenti funzioni:
  • setup():
    • si occupa di inizializzare le variabili della struct modificandole con valori a tua scelta coerenti con il tipo delle variabili
    • aggiunge segnalazioni a totaleSegnalazioni se il sensore non è guasto
  • statoGuasto():
    • prende la variabile guasto della struct come input
    • ritorna 1 se il sensore è guasto, 0 se non è guasto
  • calcolaTemperatura():
    • prende come input le variabili della struct singolarmente (quindi non passa la struct come input ma le sue variabili)
    • chiama la funzione statoGuasto e somma il suo output con la variabile temperatura della struct
    • se le segnalazioni sono maggiori di 3, allora aumentiamo temperatura di 0.5
    • la funzione ritorna LIMITE se temperatura è maggiore di LIMITE, altrimenti ritorna temperatura
I nomi delle variabili in input nella dichiarazione della funzione devono essere diversi (dove possibile) dai nomi delle variabili che nel main diamo come input alle funzioni.

Gli input devono essere passati by reference solo quando necessario/utile.

4. Nel main devi visualizzare in output:
  • il risultato di calcolaTemperatura dopo averlo trasformato in int
  • il valore della variabile totaleSegnalazioni
