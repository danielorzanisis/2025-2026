Scrivi un programma in C++ che rispetti le seguenti specifiche.

1. Crea una struct chiamata Studente che contenga tre variabili:
  • int numeroLibriLetti
  • double mediaVoti (sarà un numero compreso tra 0 e 10)
  • bool partecipazioneAttiva
2. Definisci due variabili globali:
  • const double VOTO_MAX = 10.0;
  • int studentiAttivi = 0;
3. Crea le seguenti funzioni:
  • inizializza():
    • si occupa di inizializzare le variabili della struct modificandole con valori a tua scelta coerenti con il tipo delle variabili
    • incrementa la variabile studentiAttivi se partecipazioneAttiva è true
  • verificaAttivita():
    • prende la variabile partecipazioneAttiva della struct come input
    • ritorna 9.0 se true, 4.5 se false
  • aggiornaMedia():
    • prende come input le variabili numeroLibriLetti e mediaVoti della struct
    • chiama la funzione verificaAttivita e somma il suo output con la variabile mediaVoti della struct, poi divide la somma per 2 e salva il risultato in mediaVoti
    • se la variabile numeroLibriLetti è maggiore di 5, allora aggiungiamo 3 a mediaVoti
    • la funzione ritorna VOTO_MAX se mediaVoti è maggiore di VOTO_MAX, altrimenti ritorna mediaVoti
I nomi delle variabili in input nella dichiarazione della funzione devono essere diversi (dove possibile) dai nomi delle variabili che nel main diamo come input alle funzioni.

Gli input devono essere passati by reference solo quando necessario/utile.

4. Nel main:
  • devi visualizzare in output il risultato di aggiornaMedia dopo averlo trasformato in int
