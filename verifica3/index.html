Conteggio dei percorsi in una griglia

Considera un piano cartesiano.
  • Il punto (0, 0) si trova in basso a sinistra.
  • Il punto (x, y) si trova in alto a destra, con x ≥ 0 e y ≥ 0.
Partendo dal punto (0,0), è possibile muoversi solo:
  • di una unità verso destra (incrementando x);
  • di una unità verso l’alto (incrementando y).
Consegna:
  • scrivi una funzione ricorsiva conta_percorsi(x,y) che restituisca il numero di percorsi diversi per arrivare dal punto (0,0) al punto (x, y) rispettando le regole di movimento;
  • calcola il numero di chiamate ricorsive per una griglia 4x4 motivando la tua risposta.
Per controllare se il tuo risultato è corretto, puoi utilizzare questo codice:

import math

def conta_percorsi_combinatorio(x, y):
  return math.comb(x + y, x)

print(conta_percorsi_combinatorio(2, 2)) # Output: 6
print(conta_percorsi_combinatorio(3, 5)) # Output: 56

Extra:
modifica la funzione conta_percorsi(x,y) in conta_percorsi(x_start,y_start,x,y) per contare il numero di percorsi dal punto (x_start,y_start) al punto (x,y).

Facoltativo se avanza tempo:
scrivi una versione non ricorsiva della funzione conta_percorsi(x,y) per risolvere il problema.
