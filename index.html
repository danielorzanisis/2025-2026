Questa è la piattaforma dove svolgerete la prova, una volta entrati qui non sarà più possibile navigare sul vostro pc. Al termine della prova potrete uscire dalla piattaforma premendo l'icona in basso a destra e inserendo la stessa password utilizzata per l'accesso. IMPORTANTE!! Prima di uscire dovrete salvare il lavoro svolto, una volta usciti dalla piattaforma tutto il codice non salvato viene perso. Nella metà sinistra dello schermo (dove state leggendo) troverete il testo della prova, mentre nella parte destra potete vedere un compilatore online per scrivere ed eseguire il vostro codice. Per salvare il file è sufficiente navigare nel file manager (icona del file sotto le 3 linee del menu) e premere sull'icona per il download. Fate attenzione a non uscire dalla pagina del compilatore o a ricaricarla perché anche in questo caso perderete il codice che avete compilato fino a quel momento. Fate tutte le prove che ritenete necessarie in modo da non avere problemi durante lo svolgimento della verifica. Per eventuali domande o chiarimenti mandatemi una mail.
