ESERCIZIO

  Scrivi un programma in C++ che rispetti le seguenti specifiche:

  1. Crea una struct chiamata Studente che contenga tre variabili:
      - int numeroLibriLetti
      - double mediaVoti
      - bool partecipazioneAttiva

  2. Definisci due variabili globali:
      - const double VOTO_MAX = 10.0;
      - int studentiAttivi = 0; NON costante, può essere modificata dalle funzioni

  3. Crea le seguenti funzioni:

      inizializza()
        - inizializza tutte le variabili della struct con valori a tua scelta
          coerenti con il tipo delle variabili
        - incrementa la variabile globale studentiAttivi se il bool è true

      f2:
        - riceve la variabile bool della struct
          // ritorna 99 se flag è true, 100 se flag è false
          // NOTA: il parametro si deve chiamare "flag", non "bool" o "valore"

      void f3(double& numeroLocale);
          // riceve la variabile double della struct (by reference)
          // divide il numero per 3
          // usa static_cast<double> se serve per evitare divisione intera
          // usa anche la costante globale PI in un’operazione a tua scelta
          // (ad esempio: numeroLocale = numeroLocale / 3 + PI / 10)

  4. Nel main:
      - dichiara una variabile di tipo Dati chiamata d1
      - chiama f1(d1) per inizializzarla
      - chiama f3(d1.<la_tua_variabile_double>) per modificarne il valore
      - chiama f2(d1.<la_tua_variabile_bool>) e stampa il valore restituito
      - stampa infine tutte le variabili della struct e anche la variabile globale contatore

  5. Regole aggiuntive:
      - usa nomi diversi per le variabili locali, di struct e di parametri
        per mostrare che hai capito che non sono la stessa cosa
      - usa almeno una costante (const)
      - usa almeno un passaggio by reference
      - mostra in output la differenza tra modificare una variabile globale
        e modificare una variabile passata per riferimento
