Controllo dell'ordine

Consegna:
  • scrivi una funzione ricorsiva che, data una lista di numeri, restituisca True se la lista è ordinata, False se non lo è;
  • calcola il numero di chiamate ricorsive che vengono effettuate per una lista di 10 cifre, cambia se la lista è ordinata o meno?
Esempio:
controllo_ordine([1,2,3,6,7,9]) -> True
controllo_ordine([1,2,3,11,7,9]) -> False

Conteggio dei numeri in una stringa

Consegna:
  • scrivi una funzione ricorsiva che, data una stringa, restituisca il numero di numeri che contiene;
  • calcola il numero di chiamate ricorsive che vengono effettuate per una stringa di 20 caratteri.
Esempio:
conta_numeri("4a55w0rd") -> 4
